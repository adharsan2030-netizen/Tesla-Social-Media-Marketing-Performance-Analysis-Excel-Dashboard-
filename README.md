# Tesla Social Media & Marketing Performance Analysis (Excel)

## 📊 Overview
This project analyzes Tesla’s social media performance across multiple platforms using Excel. It focuses on evaluating engagement, reach, and campaign effectiveness.

## 🎯 Objective
To identify which platforms, content types, and campaigns generate the highest engagement and audience interaction.

## 📁 Dataset Details
The dataset includes:
- Platform (Twitter, Instagram, YouTube,Facebook)
- Post type and content
- Likes, Shares, Comments
- Impressions and Reach
- Clicks and Engagement Rate
- Campaign names

## 🔧 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboards
- Data Cleaning & Formatting

## 📈 Key Analysis Performed
- Engagement Rate comparison across platforms
- Campaign performance analysis
- Content type effectiveness
- Reach vs Impressions analysis
- Click-through behavior insights

## 💡 Key Insights
- Instagram shows higher engagement rates compared to other platforms
- Campaigns like Cybertruck Launch generated strong interaction
- Video content performs better in terms of reach and engagement
- Higher impressions do not always lead to higher engagement

## 📊 Dashboard Features
- Interactive filters for platform and campaign
- KPI metrics (Engagement Rate, Clicks, Reach)
- Visual charts for performance comparison
